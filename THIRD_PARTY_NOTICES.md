# Third-Party Notices

Swing Sync incorporates third-party open-source software. Production build
notices are generated into `dist/THIRD_PARTY_NOTICES.txt`; this root document
records project policy, manual attributions, and reference-derived exceptions.

## Manual Reference Notices

No reference repository code has been copied or adapted into Swing Sync at this
time.

If permissively licensed reference code is later adapted, the pull request must:

- identify the source repository, file, and license;
- preserve original copyright and license notices in the adapted file when
  required;
- add the relevant notice here; and
- receive maintainer approval before merge.

Unlicensed or copyleft reference repositories are clean-room only and must not be
listed here as copied or adapted code.
